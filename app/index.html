Website Administration

Overview

Welcome to your custom user guide

This guide will teach you how to manage and administer http://www.acdbio.com.

Use the navigation menu on the left to skip to different categories throughout the guide.

Logging into the Dashboard

To log into the dashboard of your website, point your browser to http://www.acdbio.com/system/ and use the login credentials you were provided.

Note: Never give out this password to unauthorized users! If you change your password in the future, please make sure you use a strong password.

Introduction to Expression Engine

Acdbio.com is powered by a Content Management System (CMS) called Expression Engine. From this dashboard, you will be able to add/remove content, process orders, and manage users.

Remember: You are managing a live website! Changes you make will be immediately viewable by your users, so take extra care before committing any changes.

Managing Members

Figure X.X: Viewing all members

To manage your registered members, select "Members" from the top menu in the dashboard. The most important options in this menu are:

  • View All
  • Register Member
  • User Banning

Figure X.X: Manually registering a User

Manually registering a user

1. To manually register a member, select Members > Register Members, and fill out the corresponding fields. Required fields are marked with a *.

Figure X.X: Using Profile:Edit to import a user to the customer base

2. The user then must be imported into the system as a customer which takes a few additional steps. Select Add-Ons > Modules from the top-menu. From the list on the left, select "Profile:Edit".

Figure X.X: Importing a user to the customer base

3. After selecting "Profile:Edit" from the modules page, select "Utilities near the top of the page. On this page, select "Start" next to "Create Profile entires for existing members who do not already have a Profile" and the system will import all of the members into your customer base who have not already been imported.

Moderating Comments

Figure X.X: Moderating Comments

Moderating comments on your blog is very simple. Select "Comments" from the top menu in the dashboard. From this page you can view all of the comments, and quickly change the status of comments by selecting the check box on the right, changing the status in the dropdown menu and clicking "Submit."

Clearing the Cache

Figure X.X: CE Cache Home

ACDbio.com uses an advanced caching system to rapidly deliver pages to users by saving a version of the page, rather than rebuilding each page everytime a user views the website. However, when you are making changes on the website, you won't be able to see the changes reflected until the server has created an updated version of the page in the cache. In order to access the system to clear the cache, select Add-ons > Modules from the top menu in the dashboard. Then, select "CE Cache from the list of Add-ons. Alternatively, you can use this link.

Figure X.X: Partially clearing the cache to reflect your changes

When deciding to clear the cache, generally you only want to clear the portion of the site you would like to refresh. To clear only parts of the cache, select "View Items" next to the "File" Driver. Then, select the "Local" folder. Here you will see a list of the temporarily cached files. From here you want to find the folder of the section you are trying to clear (for instance, /about/blog/) and select "Delete Children." This will delete the cached files for that section that will then be re-generated the next time a user loads that page.

Note: The entire cache can be cleared from the CE-Cache home by selecting "Clear entire Cache for All Drivers" then "Yes I'm Sure, Clear all Driver Caches." Remember, this will clear the cache for the entire website, and all pages will not be initially cached!

Managing the Content

Blog

Figure X.X: Adding and Editing Blog Posts

To add a new blog post, select Content > Publish > Blog. To edit a blog post that has already been posted select Content > Edit > Blog. When publishing a new post fill out the Title, Blog Title, and add the copy of the post into the "Body". The text of the post can be manipulated using the buttons above the WYSIWYG.To directly edit the HTML of the post, select "Source". Other features include bold, italics, paste from word, adding links, super/sub-scripting, and many more.

The date of the post can be changed to a previous or future date using the "Date" tab. To remove a post without deleting it from the system, select the "Options" tab, set the "Status" to "closed" and then press submit.

A note about "Titles" in ExpressionEngine: You may notice that many types of content have a "Title" and a "Blog Title" or "News Title". ExpressionEngine requires every post to have a unique "Title" that it uses to identify each post in the system. The other title on the page is what will be displayed on the page.
